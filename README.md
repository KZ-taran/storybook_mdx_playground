# NextJS app with TypeScript and Storybook and MDX

- Storybook
  - https://storybook.js.org/
- MDX
  - https://mdxjs.com/

## How to use

```bash
git clone https://github.com/KZ-taran/storybook_mdx_playground.git
# and
yarn
```

### Run NextJS and Storybook

```bash
yarn dev
# and
yarn storybook
```
